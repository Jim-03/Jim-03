### Salio - Personal Finance Tracker _(Development paused April 2026)_

**Stack:** React Native (Expo) · Gaussian Naive Bayes · SQLite

A mobile app that parses M-Pesa SMS messages and automatically categorises transactions using a machine learning classifier. No manual input required - point it at your inbox and it builds your spending history.

- M-Pesa SMS parsing with regex-based extraction
- Gaussian Naive Bayes transaction classifier (trained on local spend categories)
- Batch processing optimised from ~10 seconds → 4.2 seconds
- Offline-first architecture
